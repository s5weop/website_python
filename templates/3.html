Таблица функций в Python


Функция Перевод Назначение Пример
print() Печать Выводит или печатает информацию на экран print("Hello world") - напечатает на экран надпись Hello world
input() Ввод 1. Выводит на экран подсказку, что нужно ввести пользователю.
2. Возвращает то, что пользователь ввел с клавиатуры.
3. Всегда возвращается строковый "тип ланных" 		number = input("Введите число")
сначала на экран выведется надпись Введите число
потом появится поле куда можно ввести текст с клавиатуры, далее введенный текст сохраняется в переменную number
int() Целое Меняет тип данных получаемого значения на целочисленный number = int(input("Введите число"))
возвращаемое значение с клавиатуры будет преобразовано в целочисленный тип данных и сохранено в переменную number
т. е. число 5 полученное при помощи input будет восприниматься Python-ом в кавычках "5", a int эти кавычки уберет
str() Строковое Меняет тип данных получаемого значения на строковый Если в программе нужно объединить строки и числами в одну строку
name = str(5) + "_Иван"
в результате в переменную name сохранится "5_Иван"
len() Длина Возвращает длину последовательности string = "Hello world"
result = len(stroka) -> в переменную result сохранится число 11
range() Диапазон Возвращает последовательность от 0 до заданного числа, не включая само число, но так же можно указать начальное число range(5) -> вернет 0, 1, 2, 3, 4
*range(2, 5) -> вернет 2, 3, 4 - возможный вариант использования
