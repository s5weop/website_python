Конечно! Вот пример программы на Python, реализующей метод прямоугольников для численного интегрирования:

```python
def rectangular_integration(f, a, b, n):
    dx = (b - a) / n
    integral = 0
    for i in range(0, n):
        integral += dx * f(a + i * dx)
    return integral
```

Вы можете использовать эту функцию, передавая в неё подынтегральную функцию `f`, пределы интегрирования `a` и `b`, и количество разбиений `n`.
